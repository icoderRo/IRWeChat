# LCWeChat
## 有需要的小伙伴,可以参考思路
### * 实现了 发送文本、富文本、复制粘贴、发送图片(PhotoKit 图片选择器 图片浏览器)、发送语音(录制音频mp3格式 播放音频)、发送定位、拍照
* 高仿最新QQ侧滑效果
* 高仿朋友圈,文本收缩, 下拉刷新
* 封装设置界面
* FMDB 存储model 音频录制与播放的封装
